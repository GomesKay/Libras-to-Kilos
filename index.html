CONVERSOR DE LIBRAS PARA QUILOGRAMAS

Olá!

Neste projeto será apresentado um Conversor de Libras para Quilogramas. Para quem já chegou a frequentar
academia sabe que algumas anilhas apresentam 45lbs ou 22lbs.

Então este projeto veio com a ideia de entender mais sobre a conversão e que você não se confunda tanto
quando ver uma anilha ou qualquer objeto com o sistema de peso de libras.

Faça a conversão de libras para quilos utilizando a divisão com o valor de 2,205
e de quilos para libras é multiplicando o mesmo valor.
